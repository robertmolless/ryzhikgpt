# Рыжик 3D: Тайна загородного дома — v7

Это отдельная 3D-версия игры на WebGL/Three.js для GitHub Pages и Telegram WebApp.

## Важно
Для реального 3D используется Three.js через CDN:
`https://unpkg.com/three@0.160.0/build/three.module.js`

Без внешних библиотек сделать полноценный 3D-проект в браузере можно только на чистом WebGL, но это будет сильно дольше и сложнее.

## Что есть
- 3D low-poly мир
- объёмный Рыжик
- 10 объёмных персонажей
- дом, сарай, теплица, пруд, деревья, цветы, дорожки
- свет, тени, туман, время суток
- мобильный джойстик
- инвентарь с иконками
- применение предметов рядом с NPC/локацией
- музыка через Web Audio: аккорды, бас, мелодия, ambient
- Telegram WebApp ready

## Обновление GitHub
Распакуй ZIP и загрузи файлы поверх старых. Это заменит 2D-версию на 3D-версию.
