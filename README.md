# Рыжик 3D: Тайна загородного дома — V8 Ultimate

Большая 3D-версия игры для GitHub Pages и Telegram WebApp.

## Важно
V8 использует Three.js через CDN:
`https://unpkg.com/three@0.160.0/build/three.module.js`

Это нужно для настоящего 3D в браузере. Игра всё равно не использует внешние ассеты: модели, мир, персонажи и предметы создаются кодом.

## Что добавлено в V8
- Улучшенный 3D low-poly мир.
- Более живой Рыжик: анимация хвоста, лап, дыхания, ушей, idle.
- Более детальные NPC: тело, волосы, аксессуары, маршруты, idle-анимации.
- Динамическое время суток.
- Погода, туман, дождь, гроза.
- Свет, тени, фонари, свет окон, костёр.
- Улучшенная музыка через Web Audio: аккорды, бас, мелодия, ambient.
- Звуки шагов, мяу, предметов, квестов.
- Инвентарь с применением предметов.
- Мини-игры: рыбалка, светлячки, мяу-концерт.
- Достижения.
- Сохранение localStorage.
- Telegram WebApp ready.

## Как обновить GitHub Pages
1. Распакуй ZIP.
2. Загрузи файлы поверх старых в репозиторий.
3. Commit changes.
4. Подожди 30–60 секунд.

## Если игра не открывается в Telegram
Проверь:
- GitHub Pages работает по HTTPS.
- В BotFather указан домен GitHub Pages.
- iPhone/Android не блокирует WebGL.
